## 共通事項
すべての国で、毎ターン政策ポイント `+5`。戦闘解除後に限り本籍武将数2以下で追加 `+3`

## 表の見方
以上における武将数、本籍武将数は、以下の違いがあります。

| 呼称 | AI武将 | 国に最初からいる武将 | 他国から来た援軍 | 他国へ行った援軍 |
|---|---|---|---|---|
| 武将数 | | ○ | ○ | |
| 本籍武将数 | | ○ | | ○ |

取得から一定ターンの間のみ有効な政策は、取得の翌月より起算します。
