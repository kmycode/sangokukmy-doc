## 城壁の兵種
城壁の兵種は、その都市の技術によって変化します。  
AからDになるほど、守兵の攻撃力が上がり、城壁を攻められたときに突破されにくくなります。

## 城壁兵種一覧
| 種類 | 属性 | 攻撃力 | 防御力 | 技術 | 備考 |
| -- | -- | -- | -- | -- | -- |
| 守兵A | 壁 | 20 | 30 | 0 | すべての都市の初期状態 |
| 守兵B | 壁 | 40 | 40 | 300 | |
| 守兵C | 壁 | 60 | 50 | 600 | |
| 守兵D | 壁 | 90 | 60 | 999 | |
