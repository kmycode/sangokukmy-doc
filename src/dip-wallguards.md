## 城壁の兵種
城壁の兵種は、その都市の支配国が採用する政策によって変化します。  
AからDになるほど、守兵の攻撃力が上がり、城壁を攻められたときに突破されにくくなります。

## 城壁兵種一覧
| 種類 | 攻撃力 | 防御力 | 政策 | 備考 |
| -- | -- | -- | -- | -- |
| 守兵A | 30 | 10 | | すべての都市の初期状態 |
| 守兵B | 50 | 20 | 攻防の礎 | |
| 守兵C | 70 | 30 | 土塁 | |
| 守兵D | 90 | 40 | 石城 | |
